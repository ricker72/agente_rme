## 2. Architecture

```
UI  (ui/app.py, ui/main_window.py, widgets...)
 │
 ▼
 Services  (ui/services/*.py)   ← interfaces only
 │
 ▼
 Adapters  (ui/adapters/*.py)   ← interfaces only
 │
 ▼
 Core      (FROZEN — NOT TOUCHED)
```

- **Strict layering enforced.** No import from any frozen module (`core/`, `agents/`, `architect/`, etc.).
- All data crossing the boundary uses DTOs (`ui/models/`).
