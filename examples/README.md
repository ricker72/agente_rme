# Ejemplos de Generadores RME

Este directorio contiene ejemplos de scripts Lua creados con la API real de OpenTibiaBR RME.

## Archivos

- `island_generator.lua` — Generador de isla con borde y decoración.
- `cave_generator.lua` — Generador de cueva procedural usando `noise.simplex()`.
- `temple_generator.lua` — Generador de templo con cámara y sala central.
- `city_generator.lua` — Generador de ciudad con carreteras y muros.
- `spawn_generator.lua` — Ejemplo de spawns y puntos de aparición.
- `issavi_roshamuul_hybrid.lua` — Generador híbrido de Issavi + Roshamuul.

## Uso

1. Abre RME y carga un mapa vacío o existente.
2. Copia el contenido del `.lua` deseado.
3. Ve a `Tools > Run Script` y ejecuta el script.
4. Ajusta el punto de origen o los IDs si necesitas personalizar el entorno.
